1.2 Challenge Prepend * to every non-blank line

1.2.1 Solution

Run the macro 6 times since there are six non blank lines after the first one. 

;; Keyboard Macro Editor.  Press C-c C-c to finish; press C-x k RET to cancel.
;; Original keys: C-a * C-e M-f

Command: last-kbd-macro
Key: none

Macro:

C-a			;; move-beginning-of-line
*	  		        ;; self-insert-command
C-e			;; move-end-of-line
M-f			;; forward-word

1.3 Challenge Satisfy the Go linter

1.3.1 Solution

C-M-d to go down into the parens, C-n for next line and then M-m to place the cursor at the correct indentation point 

;; Keyboard Macro Editor.  Press C-c C-c to finish; press C-x k RET to cancel.
;; Original keys: M-@ C-w // SPC C-y SPC TODO RET TAB C-y C-n M-m

Command: last-kbd-macro
Key: none

Macro:

M-@			;; mark-word
C-w			;; kill-region
//			;; self-insert-command * 2
SPC			;; self-insert-command
C-y			;; yank
SPC			;; self-insert-command
TODO			;; self-insert-command * 4
RET			;; newline
TAB			;; indent-for-tab-command
C-y			;; yank
C-n			;; next-line
M-m			;; back-to-indentation

        Version string
        Debug bool
